#  Sistema de Análisis Automatizado de Dossiers Inmobiliarios

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)

Sistema inteligente que automatiza la extracción y análisis de datos de documentos inmobiliarios en PDF, transformando información no estructurada en insights accionables.

## Características Principales

•⁠  ⁠ *Extracción Automática*: Reconocimiento inteligente de precios, metros, habitaciones, zonas y estados
•⁠  ⁠ *Dashboard Interactivo*: Visualización en tiempo real con filtros avanzados y gráficos
•⁠  ⁠ *Base de Datos Histórica*: Almacenamiento persistente con tracking de cambios
•⁠  ⁠ *Procesamiento por Lotes*: Análisis simultáneo de múltiples documentos PDF
•⁠  ⁠ *Análisis Avanzado*: Estadísticas, tendencias y comparativas automáticas

## Tecnologías Utilizadas

•⁠  ⁠*Backend*: Python, PDFPlumber, Pandas, SQLite
•⁠  ⁠*Frontend*: Streamlit, Plotly
•⁠  ⁠*Procesamiento*: Regex, OpenPyXL
•⁠  ⁠*Base de Datos*: SQLite con historial de cambios
